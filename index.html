IMPORTANT: Not a Medical Device.
This application is an educational prototype developed for learning purposes only. It is not intended for medical diagnosis, treatment, or advice. Always consult a qualified healthcare professional for medical concerns.

This Privacy Policy describes how NeuroScan AI ("we", "our", or "us") handles data. As this is an educational project built to demonstrate machine learning capabilities in medical imaging, we prioritize user privacy and data minimization.

1. No Personal Health Information (PHI) Stored

We respect the sensitivity of medical data.

  • Local Processing: All MRI scan analysis is performed locally on your device (or strictly in volatile memory for demonstration). We do not save, upload, or store your MRI images on any external server.
  • Zero Logging: We do not log or track identifiable patient information.

2. Information We Collect

Since this is a portfolio project, data collection is minimal:

  • Usage Data: We may collect anonymous crash reports to fix bugs in the application code.
  • Uploaded Images: Images you select for analysis are processed temporarily and discarded immediately after the result is displayed.

3. Technology Stack

This application relies on the following technologies for educational demonstration:

  • TensorFlow Lite / PyTorch Mobile: For on-device machine learning inference.
  • React Native / Flutter: For the mobile interface.

4. Permissions

The app requires the following permissions to function:

  • Camera/Gallery Access: Strictly to allow you to select an MRI image for analysis. We do not access your photos in the background.

5. Changes to This Policy

We may update our Privacy Policy from time to time. Thus, you are advised to review this page periodically for any changes.
